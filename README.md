Desktop app created in Python (PyQT5) for removing chosen elements from photos. App includes the extention which allows users to connect to their Stable Diffusion in order to perform better and more personalized inpainting.

User can choose inpainting method from given in the app and also can use creativity and have fun with Stable Diffusion extention, which can be set up in app's settings.

Short presentation is available in mp4 video in repo.
